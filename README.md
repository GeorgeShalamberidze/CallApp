# CallApp

## Built with: 
 - React
 - TS
 - Node
 - Express
 - AntD

### Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/GeorgeShalamberidze/CallApp.git
   ```
2. Install packages
   ```sh
   code.
   cd frontend
   npm install & npm start
   cd..
   cd backend
   npm install & npm run nodemon
   ```
